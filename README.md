# Simple E-Commerce Website Nextjs Frontend Example

This is an example of an E-Commerce website frontend. This is the frontend that our E-Commerce sample project uses.

### Full Project Tech Stack
* Spring Boot (Backend only)
* Django (Backend only)
* Next.js (Frontend only) (This repository)

This is the main tech stack that this E-Commerce platform uses, however we also use technologies such as Stripe and Google Cloud. Feel free to download this repository to check out how it works.
